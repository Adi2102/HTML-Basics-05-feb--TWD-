5. Table Tag

An HTML table consists of one 'table element' and one or more 'tr', 'th', and 'td' elements.
1) tr : element defines a table row
2) th : element defines a table header
3) td : element defines a table cell

7. Button Tag

The 'button' HTML element represents:
1) a clickable button
2) used to submit forms or anywhere in a document for accessible
3) standard button functionality.
Inside a 'button element' you can put text (and tags like i, b, strong, br, img, etc.). That is not possible with a button created with the 'input element'.
NOTE: Always specify the 'type' attribute for a 'button element', to tell browsers what type of button it is.


8. Image Tag

The 'img tag' is used to embed an image in an HTML page.
The 'img tag' has two required attributes:
1) src - Specifies the path to the image
2) alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed
